# Keystroke Logging Demonstration Project

## Description
This project demonstrates a basic keystroke logging mechanism using Python.
It captures keyboard events such as key press, key hold, and key release events.

The project also includes a simple graphical user interface (GUI) built using
Tkinter to start the keylogging process.

## Tools and Technologies
- Python
- pynput library
- Tkinter GUI

## Output Files
- logs.txt : Stores keystrokes in text format
- logs.json : Stores keystrokes in structured JSON format

## Purpose
This project is developed strictly for educational and cybersecurity awareness
purposes as part of the VOIS Edunet Program.
